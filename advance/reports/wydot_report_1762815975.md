# WYDOT Answer Report

**Question:** compare and contrast soil gradation requirements

**Answer:**

Both sets of requirements presented relate to aggregate material gradation, but they differ significantly in their approach, application, and the specific parameters they define.

**Similarities:**
*   **Purpose:** Both Table 803.12.2-1 and Table 803.14-1 define quantitative limits for aggregate gradation.
*   **Sieve-Based:** Both systems rely on the concept of sieve sizes to separate and categorize aggregate particles.
*   **Material Characterization:** Both are used to characterize and ensure the quality of aggregate materials based on their particle size distribution.
*   **Tabular Format:** Both present the information in a clear, tabular format.

**Differences (Contrast):**

1.  **Scope and Material Type:**
    *   **Table 803.12.2-1:** Specifically defines gradation requirements for "Maintenance Stockpiles (Type B)." This implies a specific application or material type for maintenance activities.
    *   **Table 803.14-1:** Defines general "Gradation Requirements: Minimum and Maximum Aggregate Size" for different "Size Classes" (Class I through VIII). This is a broader classification system for diverse aggregate applications.

2.  **Parameters Used for Gradation:**
    *   **Table 803.12.2-1:** Explicitly lists specific sieve sizes (e.g., ¾ in [19 mm], ½ in [12.5mm], ⅜ in [9.50 mm], No. 4 [4.75 mm], No. 200 [75 µm]). For each sieve, it provides "Minimum" and "Maximum" percentages of material passing. It also includes a nominal 1 in [25.0 mm] sieve with 100% passing.
    *   **Table 803.14-1:** Uses percentile values to define gradation: d15 (15% passing), d50 (50% passing), d85 (85% passing), and d100 (100% passing, which also represents the maximum aggregate size). For each percentile value, it provides "Minimum" and "Maximum" size limits in inches [mm].

3.  **Nature of Requirement:**
    *   **Table 803.12.2-1:** Sets direct numerical thresholds (min/max percentage passing) for material intended for "Maintenance Stockpiles (Type B)." This ensures the material has a consistent particle size distribution for that specific purpose. The plasticity index of no greater than three is also a unique material property requirement for this type.
    *   **Table 803.14-1:** Defines characteristic particle sizes based on percentile ranges. It establishes "Size Classes" that aggregate can adhere to. It does not dictate that a specific material *must* fall into a certain class, but rather provides criteria for classification.

4.  **Flexibility vs. Specificity:**
    *   **Table 803.12.2-1:** More rigid for the "Maintenance Stockpiles (Type B)," requiring material to fall within specified percentage ranges for *all* listed sieves.
    *   **Table 803.14-1:** More flexible in allowing for different percentile ranges within each size class, providing a broader definition for each category.

5.  **Use Case:**
    *   **Table 803.12.2-1:** Likely used to ensure aggregate suitability for specific maintenance activities where a tight gradation is desired for performance, stability, or workability.
    *   **Table 803.14-1:** Used for general aggregate classification, allowing engineers to select an aggregate that meets certain percentile characteristics for a given project, or to describe an existing aggregate. This is fundamental for specifying aggregate for various construction applications.

In essence, Table 803.12.2-1 provides a direct quality control check for a specific aggregate type, ensuring it meets definitive limits. In contrast, Table 803.14-1 offers a framework for categorizing any aggregate based on its statistical particle size distribution, serving as a guideline for material selection and specification.
